What This Project Does

  • Fetches monthly retail data from U.S. Census Bureau API
  • Cleans and transforms data with feature engineering
  • Trains Facebook Prophet model with custom holidays
  • Generates 6-month forecasts with confidence intervals
  • Explains WHY predictions fail (COVID, holidays, trends)

Tech Stack

  • Data: U.S. Census Bureau MRTS API
  • Pipeline: Python, Airflow, Azure Data Factory
  • ML: Facebook Prophet, pandas, NumPy
  • Visualization: Power BI, Plotly
  • Hosting: Azure Blob → Static fallback

Key Insight: Why Predictions Fail

Unlike typical forecast dashboards, this project analyzes and explains forecast errors:
